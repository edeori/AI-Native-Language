# Tanulságok és finomítások

Ez a dokumentum az elvégzett retrospektívek eredményeit gyűjti. Minden bejegyzés tartalmazza:
- Mikor volt a retrospektív (dátum + projekt fázis)
- Mi volt a megfigyelés
- Mit változtattunk a plugin promptjaiban / heurisztikáiban
